# Debt Feature Area

Debt management is the first-class feature. Located in:
- `src/features/debts/` — components, hooks, services
- `src/domain/debt/` — types and business logic

Status values: `active`, `paid`, `overdue`, `settled`

MVP scope:
- List debts grouped by status
- Create/edit/delete via sheet
- Partial payment tracking
- See `docs/debt-scope.md` for full details
