# Money Flow — Product Vision

A single-user personal finance web app focused on:
- **Debt management** (first-class feature)
- Cashback tracking (future)
- Personal income & expense tracking (future)
- Google Sheets sync (future)

Core principles:
- Mobile-first responsive UX
- Fast interactions: modals/sheets close immediately on submit
- Optimistic UI preferred
- No friction inside drawers or sheets

Scope boundaries:
- Single user only — no team/sharing features
- No investment portfolio tracking
- No budgeting categories beyond basic income/expense
- Cashback tracking is per transaction, not auto-discovery
